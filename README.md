# wmControl
WM HighFinesse WS8 control via ethernet

# Installation instructions
## Linux
```
python3 -m venv env  # Create a virtual environment for the build tools
pip install -r requirements.txt
pre-commit install
```

## Windows
```
conda install pip
pip install -r requirements.txt
pre-commit install
```
